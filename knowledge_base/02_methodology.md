# Методика проведения занятий

Методика центра построена на принципе «от задачи к теории»: ученик сначала
сталкивается с проблемой, а затем осваивает инструмент для её решения.

## Структура типового занятия

1. Разминка — короткая задача на 5–10 минут для входа в работу.
2. Разбор темы — объяснение нового материала с примерами.
3. Практика — самостоятельное решение задач под наблюдением тренера.
4. Разбор решений — совместный анализ ошибок и удачных подходов.
5. Домашнее задание — закрепление темы.

## Принципы работы тренера

- Не давать готовый ответ сразу, а наводящими вопросами подводить ученика к решению.
- Поощрять самостоятельный поиск и эксперименты.
- Давать обратную связь по коду: читаемость, корректность, эффективность.
- Поддерживать комфортную атмосферу: ошибка — это часть обучения.

## Планирование курса

Курс делится на модули по 4–6 занятий. Для каждого модуля заранее определяются
цели обучения, список тем и набор задач возрастающей сложности. Тренер ведёт
журнал посещаемости и прогресса учеников.
